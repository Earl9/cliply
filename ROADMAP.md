# Roadmap

This roadmap is intentionally conservative while Cliply is preparing for
`v0.4.0-beta.1`.

## Current Focus: v0.4.0-beta.1 Stabilization

- Complete manual installer matrix testing.
- Complete real WebDAV, FTP, and FTPS testing.
- Verify automatic paste in real Windows applications.
- Verify DPI and multi-monitor behavior.
- Re-run log sampling after manual workflows.
- Prepare GitHub repository documentation and templates.
- Sign release binaries when possible.

## Near Term

- Expand regression tests for paste failure modes.
- Expand regression tests for sync rollback and corrupted package handling.
- Improve release automation and artifact checksums.
- Document troubleshooting for WebDAV/FTP/FTPS providers.
- Add clearer screenshots for the public GitHub README.

## Later

- Harden multi-device conflict reporting.
- Improve diagnostics export while preserving privacy.
- Add more provider-specific sync guidance.
- Evaluate optional localization improvements.
- Evaluate stable update distribution after signed builds are available.

## Non-Goals For The Current Beta

- Hosted Cliply cloud service
- Account system
- AI/OCR features
- Plugin marketplace
- Cross-platform feature parity beyond the current Windows-first target
