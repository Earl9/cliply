## 项目概况

Cliply 是 Windows 优先的 Tauri v2 + React + TypeScript + Vite 剪贴板管理器。

主要目标：

- 本地优先，不引入云服务和账号系统
- Windows MVP 可运行
- 支持剪贴板文本、链接、代码、图片历史
- 支持本地 SQLite 存储
- 支持加密同步包和远程存储 provider 架构
- 当前远程同步已实现 Local Folder、WebDAV、FTP/FTPS
- 不做 AI、OCR、插件市场、账号系统

## 数据位置

安装目录只放程序文件。日志、数据库、blob 等本地数据不放安装目录。

常见位置由 Tauri app data 目录管理，一般在：

```text
%APPDATA%\com.cliply.app\
```

可在设置页“关于”中查看实际数据目录、日志目录和数据库文件路径。
