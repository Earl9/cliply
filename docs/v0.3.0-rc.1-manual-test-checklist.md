# Cliply v0.3.0-rc.1 发布前手动测试清单

## 范围

本轮只验证发布稳定性：日志、诊断、错误提示、性能风险、同步可靠性、安装/覆盖安装。不新增 AI、OCR、账号、云同步或插件市场。

## 剪贴板核心功能

- 启动 Cliply，确认窗口 2 秒内可见，列表可加载。
- 复制普通文本、链接、代码片段，确认列表新增记录。
- 连续复制同一内容，确认开启“忽略重复内容”后只更新原记录时间/次数。
- 暂停监听后复制内容，确认不新增；恢复监听后再次复制可新增。
- 删除一条未固定记录，确认列表消失，重启后仍不显示。
- 固定/取消固定记录，确认状态立即更新，重启后保留。
- 搜索中文、英文、链接关键字，确认列表响应且不明显卡 UI。
- 切换全部/文本/链接/图片/代码/固定筛选，确认结果真实变化。

## 粘贴功能

- 在 Notepad 中选中文本项双击或回车粘贴，确认内容进入目标窗口。
- 使用“无格式粘贴”，确认只写入纯文本。
- 对图片项执行复制/粘贴，确认剪贴板中是图片。
- 目标窗口不可粘贴或焦点丢失时，应显示“目标窗口无法自动粘贴，内容已尽量复制到剪贴板”。
- 粘贴失败日志只允许出现 item_id、模式和错误类型，不允许出现剪贴板正文。

## 图片功能

- 复制截图或图片，确认左侧有缩略图，右侧详情可查看大图。
- 删除图片源文件或模拟路径不存在，确认显示“图片读取失败，原图可能已被移动或删除”。
- 图片缩略图通过 asset URL 异步加载，列表滚动不应明显阻塞。
- 导出同步包不包含图片原图 blob，只同步图片元数据/tombstone。

## 设置页

- 打开设置，确认 Header/Footer 固定，右侧内容区独立滚动。
- 通用、快捷键、历史记录、外观、同步、关于 Tab 可切换；设置页不再展示“隐私”Tab。
- 快捷键录制可检测 Cliply 内部冲突和常见系统冲突。
- 保存设置失败时显示可理解错误，不应只显示 SQLite/系统原始异常。
- 关于页显示版本、数据目录、日志目录、数据库大小、历史记录数量、最近同步时间/状态、最近错误。
- 点击“打开日志目录”能打开系统文件夹。
- 点击“复制诊断信息”能复制文本，且不包含剪贴板正文、同步密码、token/private key。
- 历史记录 Tab 可设置保存图片、保存 HTML 富文本、忽略重复内容、最大历史条数和自动清理天数。

## 主题

- 切换推荐主题，确认主窗口和设置页 token 同步变化。
- 自定义强调色，确认按钮、选中态、badge/focus 边框使用新颜色。
- 恢复方案色后，预览与实际 UI 一致。
- 深色/跟随系统当前如为禁用态，应不可点击或有明确状态。

## 同步

- 设置同步密码并保存，关闭设置页再打开，确认“已保存密码”状态仍存在，但输入框不回填明文。
- 本地文件夹 provider：选择目录后保存，执行立即同步，确认 `CliplySync/manifest.json` 和 `snapshots/*.cliply-sync` 生成。
- FTP/FTPS provider：保存配置后立即同步，确认远端目录结构为 `CliplySync/manifest.json`、`snapshots/`、`events/`、`devices/`。
- WebDAV provider：地址、用户名、密码为空或地址格式错误时显示明确提示。
- 错误同步密码导入 `.cliply-sync`，应失败并提示“同步密码错误或同步包已损坏”。
- 损坏同步包导入，应提示格式错误或文件损坏。
- 导入中发生错误，确认事务回滚，历史记录没有半导入脏数据。
- 两台设备指向同一同步目录：A 复制 3 条并固定 1 条同步，B 立即同步后应恢复 3 条且固定状态保留。
- 自动同步开启后等待一个同步间隔，确认无需手动点击也会尝试同步并更新最近状态。

## 安装器

- 全新安装到默认目录，确认开始菜单快捷方式、可选桌面快捷方式、开机自启选项生效。
- 安装到自定义目录，第二次启动安装器应识别旧目录并显示更新模式。
- 覆盖安装时本地 AppData 数据库和设置不被删除。
- 安装过程中不应弹出可见黑色命令窗口。
- 卸载默认只删除程序文件，勾选“删除本地历史记录与设置”时才清理用户数据。

## 卸载/覆盖安装

- Cliply 正在运行时执行覆盖安装，应能关闭进程或提示退出后重试。
- 覆盖安装后任务栏图标、托盘图标、窗口图标一致。
- 卸载后开始菜单、桌面快捷方式、开机自启项被清理。
- 卸载后重新安装，旧数据是否保留应与卸载页勾选一致。

## 多显示器 / DPI

- 100%、125%、150% DPI 下主窗口无文字重叠，设置页按钮不溢出。
- 多显示器切换主屏后快捷键打开窗口位置合理。
- 最小化、关闭到托盘、托盘恢复在多显示器环境中可用。

## 当前已知问题

- 当前版本仍以 Windows MVP 为主，非 Windows 剪贴板监听只是 adapter fallback。
- 深色主题/跟随系统仍属于占位或未完整实现。
- 同步包暂不同步图片原图 blob，只同步可同步元数据和文本格式。
- WebDAV、FTP/FTPS 已有 provider；SFTP、S3/R2 仍是预留，不可真实连接。
- 自动同步依赖本机已保存同步密码；密码不会明文回填到设置输入框。
- NSIS/MSI 仍可作为 fallback；Modern Installer 才是主安装体验。

## v0.3.0-rc.1 发布前阻塞项

- 真实 Windows 安装包验证：全新安装、覆盖更新、自定义路径识别、卸载保留数据。
- 真实 FTP/FTPS 或 WebDAV 环境验证：立即同步和自动同步至少各跑一轮成功/失败路径。
- 错误同步密码导入验证：失败后数据库记录数不变。
- 日志抽样检查：不得出现剪贴板正文、同步密码、Authorization、token/private key、图片内容或长加密 payload。
- 高 DPI 截图检查：主窗口、设置页、关于页不出现裁切、重叠和不可点击区域。
